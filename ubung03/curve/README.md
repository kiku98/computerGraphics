# CG1 Tutorial Example: Bézier Curve

This code skeleton demonstrates Bézier Curve and the de Casteljau algorithm.

## Usage

1. Install Node.js

2. Install dependencies for this example:

```sh
npm install
```

3. Run this project:

```sh
npm start
```
