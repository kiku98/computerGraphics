# CG1 Tutorial Example: Cameras

This code skeleton demonstrates the two basic types of cameras
in [three.js](https://threejs.org): Perspective and orthographic camera.

## Usage

1. Install Node.js

2. Install dependencies for this example:

```sh
npm install
```

3. Run this project:

```sh
npm start
```
